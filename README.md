# COVID-19-TimeSeries

#### Description
The main focus of this project is to forecast cumulative confirmed cases for Canada, as a whole and for the following provinces: Alberta, British Columbia, Ontario and Quebec. The data is collected from the publicly available information on confirmed and presumptive positive cases during the ongoing COVID-19 outbreak in Canada.
